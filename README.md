Esse é um projeto console que grava o audio de um video do Youtube. Primeiro digite o nome do arquivo Mp3 ou pressione Enter para usar
um nome padrão. Depois coloe a URL do Video, precione Enter e o mp3 do Video será gravado.

This is a console project that records the audio from a YouTube video. First, type the MP3 filename or press Enter to use a default name. 
Then paste the video URL, press Enter, and the MP3 of the video will be recorded.
